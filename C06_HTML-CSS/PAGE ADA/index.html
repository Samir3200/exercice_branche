La vie d'Ada

Ada Lovelace, mathématicienne britannique XIXe siècle, est considéré comme la première programmeuse de l'histoire pour avoir écrit le premier algorithme destiné à être executé par une machine, la machine analytique de Charles Babbage, ce qui en fait une pionnière visionnaire de l'informatique moderne.

Le nom de l'école Ada Tech School s'inspire de cette mathématicienne.

En quelques points :
    • Pionnière de l'informatique : créa le premier algorithme pour ordinateur
    • Mathématicienne visionnaire du 19è siècle
